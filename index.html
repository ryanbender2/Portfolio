Minesweeper I â€” Grid by Donnamae (edabit)

Description: This challenge is based on the game Minesweeper. Create a function that takes a list of # and -, where each hash (#) represents a mine and each dash (-) represents a mine-free spot. Return a list where each dash is replaced by a digit indicating the number of mines immediately adjacent to the spot (horizontally, vertically, and diagonally).

My Solution:
    def num_grid(game_grid):
    # Find bomb locations (#)
    bomb_locations = []
    for column in range(len(game_grid)):
        for row in range(len(game_grid)):
            if game_grid[column][row] == '#':
                bomb_locations.append((column, row))
            else:
                game_grid[column][row] = 0

    # Add surrouding bomb amounts around each bomb
    for bomb in bomb_locations:
        for bomb_amounts in [(bomb[0] - 1, bomb[1] - 1),
                  (bomb[0] - 1, bomb[1]),
                  (bomb[0] - 1, bomb[1] + 1),
                  (bomb[0], bomb[1] - 1),
                  (bomb[0], bomb[1] + 1),
                  (bomb[0] + 1, bomb[1] - 1),
                  (bomb[0] + 1, bomb[1]),
                  (bomb[0] + 1, bomb[1] + 1)]:
            try:  # index error checking for edges of grid
                if -1 not in [bomb_amounts[0], bomb_amounts[1]]:
                    if game_grid[bomb_amounts[0]][bomb_amounts[1]] != '#':
                        game_grid[bomb_amounts[0]][bomb_amounts[1]] += 1
            except IndexError:
                None

    # Convert ints back to strings
    for column in range(len(game_grid)):
        for row in range(len(game_grid)):
            game_grid[column][row] = str(game_grid[column][row])

    return game_grid

    # Run solution
    print(num_grid([
      ["-", "-", "-", "#", "#"],
      ["-", "#", "-", "-", "-"],
      ["-", "-", "#", "-", "-"],
      ["-", "#", "#", "-", "-"],
      ["-", "-", "-", "-", "-"]]))

Jake's Meal Time by Helen Yu (edabit)

Description: Jake is a very habitual person. He eats breakfast at 7:00 a.m. each morning, lunch at 12:00 p.m. and dinner at 7:00 p.m. in the evening. Create a function that takes in the current time as a string and determines the duration of time before Jake's next meal. Represent this as a list with the first and second elements representing hours and minutes, respectively.

My Solution:
    def time_to_eat(current_time):
    # split time and THC
    time, THC = current_time.split()
    hr, minute = time.split(':')
    
    # change time into military format
    if THC[0] == 'p':
        time24 = str(int(hr) + 12) + ':' + minute
    else:
        time24 = time
    hr_24, minute_24 = time24.split(':')
    time_in_minutes = (int(hr_24) * 60) + int(minute_24)
    
    # the time falls through each check until it slides into correct category
    if time_in_minutes < 420:  # 7:00 a.m.
        tte_hr = int((420 - time_in_minutes) / 60)
        tte_min = abs((time_in_minutes % 60) - 60)
        if tte_min == 60:
            tte_min = 0
        return [tte_hr, tte_min]
    
    if time_in_minutes < 720:  # 12:00 p.m.
        tte_hr = int((720 - time_in_minutes) / 60)
        tte_min = abs((time_in_minutes % 60) - 60)
        if tte_min == 60:
            tte_min = 0
        return [tte_hr, tte_min]
    
    if time_in_minutes < 1140:  # 7:00 p.m.
        tte_hr = int((1140 - time_in_minutes) / 60)
        tte_min = abs((time_in_minutes % 60) - 60)
        if tte_min == 60:
            tte_min = 0
        return [tte_hr, tte_min]
    
    if time_in_minutes > 1139:  # any time after 7:00 p.m.
        tte_hr = int((1440 - time_in_minutes) / 60) + 7
        tte_min = abs((time_in_minutes % 60) - 60)
        if tte_min == 60:
            tte_min = 0
        return [tte_hr, tte_min]


    # Run solution
    print(time_to_eat("8:01 p.m."))

Bit Rotation by Albrecht (edabit)

Description: Python offers some bit operations but not bit rotation. To complete that, create a function that takes three parameters: n: Integer, which in binary representaion should be rotated. m: Number of rotation steps that should be performed. d: Boolean value; True = rotation right, False = rotation left.

My Solution:
    def bit_rotate(number, rotations, l_or_r):
    # convert number into into binary
    bin_num = bin(number)[2:]
    
    # location position to split number
    pos = len(bin_num) - rotations if l_or_r else rotations
    
    # flip bits at split position and convert back to base 10
    return int(bin_num[pos:] + bin_num[:pos], 2)

    
    # Run solution
    print(bit_rotate(58, 9, True))
